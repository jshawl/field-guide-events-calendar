A plugin to display a calendar of events from Neon CRM registrations.

== Description ==

Displays a calendar of events from Neon CRM.

== Usage ==

Add a short code to any page or post to display a calendar of upcoming events:

`[neoncrm_calendar]`

To allow users to filter events by event category, add the 
`filter_categories="true"` attribute:

`[neoncrm_calendar filter_categories="true"]`

== Installation ==

Install the plugin and configure the Org ID and Api Key in Settings > Neon CRM Calendar.

Information about the Org ID and Api Key are available on [Neon's developer site](https://developer.neoncrm.com/api/getting-started/api-keys/).
